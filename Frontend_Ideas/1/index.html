Thoughts on Atomic Habits

This book completely changed how I view progress. The idea of 1% improvements compounding over time is powerful not just for life, but for coding as well. Highly recommended for anyone looking to build better systems.

Thoughts on Deep Work

In an age of distraction, this book is a survival guide. It convinced me to shut off notifications and focus on 'deep' tasks for 4 hour blocks. My productivity doubled after applying these techniques.

Thoughts on Clean Code

A classic for a reason. While some examples are dated, the core philosophy of writing code for humans first, machines second, is timeless. It made me a much more empathetic developer.
